# Movilabs Website

Landing institucional oficial de Movilabs.

Movilabs es un estudio independiente enfocado en crear aplicaciones móviles simples, útiles y bien diseñadas.

## 🌐 Sitio web

https://movilabs.app

## 🧩 Proyecto

Este repositorio contiene la landing institucional de Movilabs, diseñada para presentar el estudio, su enfoque de desarrollo y facilitar el contacto institucional.

La página está disponible en tres idiomas:

- Español
- Inglés
- Portugués

## ⚙️ Stack tecnológico

- Next.js (App Router)
- TypeScript
- Tailwind CSS

El proyecto es una landing estática sin backend ni base de datos.
